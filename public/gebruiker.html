Registreren

Leden van de schaakvereniging kunnen zich registreren als gebruiker.

Een geregistreerde gebruiker heeft op 0-0-0.nl een persoonlijke agenda, kan zich per ronde aanmelden of afzeggen en kan de voorlopige indeling van de komende ronde zien.

registratieformulier

De registratie gaat in een aantal stappen:

Selecteer je naam

Klik op je naam in de lijst van leden van de schaakvereniging. Je naam en je KNSB nummer worden automatisch ingevuld op het registratieformulier.

Vul je e-mail in en klik op registratie aanvragen

We hebben je e-mail nodig om je een link met een unieke code op te sturen waarmee 0-0-0.nl je herkent. Indien 0-0-0.nl je niet meer herkent, kan je opnieuw een registratie aanvragen.

Je aanvraag wordt gecontroleerd

Je aanvraag wordt niet onmiddellijk verwerkt, want we controleren eerst je e-mail. Bovendien krijg je uitsluitend per e-mail een link met een unieke code. 0-0-0.nl stuurt dus nooit een unieke code. Het is de bedoeling dat het apparaat waarmee je 0-0-0.nl gebruikt, je unieke code als gebruiker automatisch naar 0-0-0.nl stuurt.

Klik op de link, die je per e-mail hebt gekregen

0-0-0.nl genereert voor elk lid van de schaakvereniging een unieke code. Die unieke code krijg je per e-mail met een link. Pas als je op die link hebt geklikt, wordt de unieke code geactiveerd en kan 0-0-0.nl je herkennen als geregistreerd gebruiker. Indien je 0-0-0.nl gebruikt op verschillende apparaten (computer, tablet of smartphone), moet je op elk apparaat dezelfde unieke code activeren.

0-0-0.nl herkent je als geregistreerd gebruiker

Als je een geregistreerd gebruiker bent, staat een unieke code op elk apparaat waarop je de registratie hebt geactiveerd. Indien je met dat apparaat 0-0-0.nl gebruikt, stuur je automatisch die unieke code naar 0-0-0.nl. Met de unieke code zoekt 0-0-0.nl je naam en KNSB nummer en heb je een persoonlijke agenda, kan je je per ronde aanmelden of afzeggen en kan je de voorlopige indeling van de komende ronde zien.
